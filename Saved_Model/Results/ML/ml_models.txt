
# EEG Seizure Classification using Machine Learning

## 📌 Project Overview
This project focuses on classifying EEG brain signals into three categories:
- Healthy
- Ictal (seizure)
- Nonictal

Machine Learning models such as Support Vector Machine (SVM) and Random Forest (RF) are used for classification.

---

## 📊 Dataset Description
- Total samples: 500
- Features: 15 EEG-based statistical and frequency features
- Classes:
  - healthy
  - ictal
  - nonictal

No missing values were found in the dataset.
------------------------------
CLASS MAPPING
------------------------------
healthy   -> 0
ictal     -> 1
nonictal  -> 2


⚙️ Methodology

1. Data Preprocessing
- Label Encoding applied to convert categorical labels into numerical form
- Feature scaling performed using StandardScaler for uniform feature range

2. Feature Selection
- Random Forest feature importance used to rank features
- Top 10 most important features selected for classification
- Entropy-based and frequency-domain features contributed most

3. Model Training
Two machine learning models were trained:

✔ Support Vector Machine (SVM)
- Kernel: Radial Basis Function (RBF)
- Suitable for non-linear classification of EEG signals

✔ Random Forest Classifier
- Ensemble learning method using multiple decision trees
- Provides robust performance and feature importance ranking
