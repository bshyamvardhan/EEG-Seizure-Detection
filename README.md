# EEG-Based Epileptic Seizure Detection

This repository contains our mini project on EEG-based epileptic seizure detection using Machine Learning and Deep Learning techniques.

## Models Used
- Support Vector Machine (SVM)
- Random Forest
- Custom CNN
- ResNet50 (Transfer Learning)

## Explainable AI
- Grad-CAM

## Dataset
- Bonn University EEG Dataset
